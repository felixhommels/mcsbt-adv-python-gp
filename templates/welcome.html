Welcome to the Food Delivery Prediction API!

This API allows you to predict food delivery times based on various factors such as distance, weather, traffic, and more.

Use the following endpoints to interact with the API:

  • Home Page: / - Displays this welcome message.
  • Prediction Page: /predict - Make predictions by providing the necessary parameters.
    • Method: POST
    • Parameters:
      • Distance_km: Distance in kilometers (float)
      • Weather_Clear: 1 if clear, 0 otherwise
      • Weather_Foggy: 1 if foggy, 0 otherwise
      • Weather_Rainy: 1 if rainy, 0 otherwise
      • Weather_Snowy: 1 if snowy, 0 otherwise
      • Weather_Windy: 1 if windy, 0 otherwise
      • Traffic_Level_Low: 1 if low traffic, 0 otherwise
      • Traffic_Level_Medium: 1 if medium traffic, 0 otherwise
      • Traffic_Level_High: 1 if high traffic, 0 otherwise
      • Time_of_Day_Afternoon: 1 if afternoon, 0 otherwise
      • Time_of_Day_Evening: 1 if evening, 0 otherwise
      • Time_of_Day_Morning: 1 if morning, 0 otherwise
      • Time_of_Day_Night: 1 if night, 0 otherwise
      • Vehicle_Type_Bike: 1 if bike, 0 otherwise
      • Vehicle_Type_Car: 1 if car, 0 otherwise
      • Vehicle_Type_Scooter: 1 if scooter, 0 otherwise
      • Preparation_Time_min: Preparation time in minutes (float)
      • Courier_Experience_yrs: Courier experience in years (float)
    • Example JSON Request: 
      {
  "Distance_km": 5.2,
  "Weather_Clear": 1,
  "Weather_Foggy": 0,
  "Weather_Rainy": 0,
  "Weather_Snowy": 0,
  "Weather_Windy": 0,
  "Traffic_Level_Low": 0,
  "Traffic_Level_Medium": 1,
  "Traffic_Level_High": 0,
  "Time_of_Day_Afternoon": 1,
  "Time_of_Day_Evening": 0,
  "Time_of_Day_Morning": 0,
  "Time_of_Day_Night": 0,
  "Vehicle_Type_Bike": 0,
  "Vehicle_Type_Car": 1,
  "Vehicle_Type_Scooter": 0,
  "Preparation_Time_min": 15.5,
  "Courier_Experience_yrs": 3.2
}
  • Statistics Route: /statistics - View statistics filtered by various query parameters.

    You can only chose one per category since it cant be sunny or rainy at the same time for example.

    • Method: GET
    • Query Parameters:
      • weather: Weather condition (e.g., "clear", "rainy", "snowy", "foggy", "windy")
      • traffic: Traffic level (e.g., "low", "medium", "high")
      • time_of_day: Time of day (e.g., "morning", "afternoon", "evening")
      • vehicle_type: Vehicle type (e.g., "bike", "car")
      • courier_experience: Courier experience in years (numeric)
    • Example Request: url/statistics?weather=rainy
  • Data Route: /data/<order_id> - Get delivery data for a specific order ID.
    • Method: GET
    • Path Parameter:
      • order_id: Integer ID of the order
    • Example Request: url/data/522
